# Changelog

All notable changes to the ASO Optimizer skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-23

### Added
- Initial release of ASO Optimizer skill
- Core keyword research functionality with semantic analysis
- Competitor analysis using web search
- Metadata optimization for iOS App Store (30/30/100 character limits)
- Metadata optimization for Google Play (30/80/4000 character limits)
- Multi-language localization support
- Comprehensive reference files:
  - `store-guidelines.md` - Platform-specific ASO best practices
  - `keyword-strategies.md` - Advanced keyword research methodologies
  - `competitor-analysis.md` - Systematic competitor research framework
- Character count validation for all metadata fields
- Integration with xlsx skill for tracking spreadsheets
- Integration with docx skill for formal reports
- Integration with web_search for competitor intelligence
- MIT License for free distribution
- Complete documentation (README, CONTRIBUTING)

### Features
- Automatic skill activation when discussing ASO
- Guided workflow from app description to final metadata
- Keyword tiering (Tier 1/2/3) based on priority
- User intent mapping (navigational, informational, transactional)
- Long-tail keyword generation
- Competitive gap analysis
- Multiple output formats (Markdown reports, Excel spreadsheets)
- ASO best practices enforcement (no keyword stuffing, competitor names, etc.)

### Documentation
- Comprehensive README.md with installation and usage instructions
- CONTRIBUTING.md with guidelines for contributors
- Example workflows for common use cases
- Troubleshooting section
- Integration guide with DOCK app
